# Task: Example Store UI - Upload & Similarity Testing (P1)

## Description

**Priority:** P1 (deferred after MVP)

Build the frontend interface for the Example Store in the TOOLS section (Alt+E). Users can create, browse, and test few-shot examples. The UI enables uploading examples, testing similarity search with sample queries, and managing example quality ratings.

**Access:** TOOLS → Example Store (keyboard shortcut: Alt+E)

## Acceptance Criteria

- [ ] React component `ExamplesPage.tsx` for Example Store management
- [ ] Form to create new example (input + expected output + metadata)
- [ ] List view showing all examples with filters (function_name, domain, quality_score)
- [ ] Similarity testing tool: enter query → see top-5 matching examples
- [ ] Edit example quality score
- [ ] Delete example
- [ ] Visual similarity score indicator (0.0 - 1.0 cosine similarity)
- [ ] Keyboard shortcut Alt+E opens Example Store

**API Integration:**
```typescript
// frontend/src/services/api.ts
export const examplesApi = {
  createExample(example: ExampleCreate): Promise<Example>
  listExamples(filters?: ExampleFilters): Promise<Example[]>
  searchExamples(query: string, filters?: SearchFilters): Promise<ExampleSearchResult[]>
  updateQualityScore(id: string, score: number): Promise<Example>
  deleteExample(id: string): Promise<void>
}
```

**TypeScript Types:**
```typescript
// frontend/src/types/examples.ts
export interface Example {
  id: string
  input: any  // JSON input
  expected_output: any  // JSON output
  search_keys: string[]
  function_name?: string
  domain?: string
  quality_score: number  // 0-1
  created_by: string
  created_at: string
}

export interface ExampleSearchResult extends Example {
  similarity_score: number  // Cosine similarity 0-1
}
```
